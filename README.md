#  Velour – AI Fashion Advisor
Velour – your AI-powered fashion companion that turns style into a statement. Discover personalized outfit recommendations using ML &amp; NLP, tailored to your vibe and occasion. With seamless Supabase authentication, Velour makes styling smart, effortless, and uniquely you.

*Style that speaks before you do.* 💅

Velour is an AI-powered fashion advisor that delivers personalized outfit recommendations based on user preferences and occasions. Built using ML (SVM & Naive Bayes) with NLP concepts and Supabase authentication for a seamless experience.


## 🌟 Features

👗 Personalized outfit suggestions
🧠 AI/ML-based recommendations
🔐 Supabase authentication
🎯 Occasion-based styling


## 🛠️ Tech Stack

* **UI:** Streamlit
* **Backend:** Python
* **Libraries:** NumPy, Pandas, Scikit-learn
* **ML:** SVM, Naive Bayes
* **Auth/DB:** Supabase


## 🚀 Run Locally

pip install -r requirements.txt
streamlit run app.py


## 🔐 Environment Variables

SUPABASE_URL=your_url
SUPABASE_ANON_KEY=your_key
